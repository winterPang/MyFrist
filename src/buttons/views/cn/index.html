Button按钮

绿洲OASIS提供一套基于绿洲的按钮组件

默认按钮 基础的按钮用法。 
html:
<oasis-button type ="button" text="默认按钮"></oasis-button>
<oasis-button type ="hover" text="默认按钮"></oasis-button>
<oasis-button type ="text" text="默认文本"></oasis-button>
<oasis-button type ="link" text="默认链接"></oasis-button>

type类型只用三个属性button、text、link表示按钮，文字按钮，链接按钮。text属性描述按钮内容

禁用按钮 按钮不可用状态。 
html:
<oasis-button type ="button" disabled text="默认按钮"></oasis-button>
<oasis-button type ="hover" disabled text="默认按钮"></oasis-button>
<oasis-button type ="text" disabled text="默认文本"></oasis-button>
<oasis-button type ="link" disabled text="默认链接"></oasis-button>

disabled属性或者class="disabled"都可以实现禁用按钮

<oasis-button icon="glyphicon glyphicon-plus" text="增加"></oasis-button>
<oasis-button icon="fa fa-edit" text="编辑"></oasis-button>
<oasis-button icon="fa fa-trash" text="删除"></oasis-button>

<oasis-button icon="glyphicon glyphicon-plus" ></oasis-button>

<oasis-button icon="fa fa-search" icon-right text="搜索"></oasis-button>
<oasis-button icon="glyphicon glyphicon-plus" icon-right text="增加"></oasis-button>
<oasis-button icon="fa fa-edit" icon-right text="编辑"></oasis-button>
<oasis-button icon="fa fa-trash" icon-right text="删除"></oasis-button>

icon属性图标字体或者自定义class都可以实现button按钮添加图标。也可以只需要图标。通过icon-right表示图标放到右侧。默认在左侧。不支持type为其他类型。

属性配置
元素 属性 类型 可选值 默认值 说明
oasis-button type string btn hover link text btn type属性描述了按钮类型，提供了四种类型的按钮：btn正常按钮类型，link链接类型按钮，悬浮按钮类型，文字按钮类型。
text string --- --- text属性按钮文本描述.。按钮文字内容
scene string warning success danger 可以赋值也可以不赋值。default scene属性描述按钮使用场景类型。常用于描述在成功、危险、警告类
disabled boolean --- --- disabled属性标示按钮禁用状态。也可以通过添加class为.disabled
icon string --- --- icon属性用于配置按钮显示的图标字体，可以采用ICON图标字体库
icon-right boolean true/false或者只配置属性 icon-right属性配置icon图标显示在文字的位置
native-type string submit button reset button native-type按钮原生type属性
