Layout布局

基于bootstrap栅格布局，进行绿洲布局实现

基本布局 
html:
<oasis-row>
  <oasis-col md="12"></oasis-col>
</oasis-row>

<oasis-row>
  <oasis-col md="6"></oasis-col>
  <oasis-col md="6"></oasis-col>
</oasis-row>

<oasis-row>
  <oasis-col md="4"></oasis-col>
  <oasis-col md="4"></oasis-col>
  <oasis-col md="4"></oasis-col>
</oasis-row>

布局组件使用oasis-row 和 oasis-col配合使用。 其中使用oasis-col元素md属性表示单元格所占比例,值只能是数字number类型。 还可以直接使用class="col-md-4",实现栅格布局。

响应式布局 
html:
<oasis-row>
  <oasis-col md="12" sm='12' lg="12" xs="12" ></oasis-col>
</oasis-row>

<oasis-row>
  <oasis-col md="6" sm='6' lg="6" xs="6" ></oasis-col>
  <oasis-col md="6" sm='6' lg="6" xs="6" ></oasis-col>
</oasis-row>

<oasis-row>
  <oasis-col md="4" sm='4' lg="4" xs="4" ></oasis-col>
  <oasis-col md="4" sm='4' lg="4" xs="4" ></oasis-col>
  <oasis-col md="4" sm='4' lg="4" xs="4" ></oasis-col>
</oasis-row>

使用了 Bootstrap 的 响应式设计，预设了四个响应尺寸：xs、sm、md和lg。 还可以直接使用class="col-md-4",实现栅格布局。

盒子布局 body 
html:
<oasis-box>
  <oasis-box-header>
    <span class="title">盒子布局</span>
  </oasis-box-header>
  <oasis-box-body></oasis-box-body>
  <oasis-box-footer></oasis-box-footer>
</oasis-box>

盒子布局，通过新的元素box box-header box-body box-footer元素组合实现共性的盒子模型布局。 box-body 设置显示内容区域高度 height = 100 height数值number类型，也可以通过style="height:100px"实现设置。

显示代码 隐藏代码
无背景色盒子 body 
html:
<oasis-box>
  <oasis-box-header  bgnone>
    <span class="title">盒子布局</span>
  </oasis-box-header>
  <oasis-box-body></oasis-box-body>
  <oasis-box-footer></oasis-box-footer>
</oasis-box>

box-header bgnone属性去掉header背景色。 或者通过style="background:transparent;"，同时支持class 属性自定义样式

元素 属性 类型 可选值 默认值 说明
oasis-row ---- ---- ---- ---- 类似Bootstrap .row 表示一行配合oasis-col一起使用
oasis-col md number ---- ---- (≥992px) 类似Bootstrap .col-md-4 中等屏幕 桌面显示器实行12格栅格布局。
例如：md=4
PS：配合oasis-box使用时，一定要添加class="app-colum left-block"。
sm number ---- ---- (≥768px)类似Bootstrap .col-sm-4 小屏幕 平板 实行12格栅格布局。
例如：sm=4
PS：配合oasis-box使用时，一定要添加class="app-colum left-block"。
lg number ---- ---- (≥1200px) 类似Bootstrap .col-lg-4 大屏幕大桌面显示器 实行12格 栅格布局。
例如：lg=4
PS：配合oasis-box使用时，一定要添加class="app-colum left-block"。
xs number ---- ---- (<768px)类似Bootstrap .col-xs-4 超小屏幕 手机实行12格栅格布局。
例如：md=4
PS：配合oasis-box使用时，一定要添加class="app-colum left-block"。
oasis-box ---- ---- ---- ---- 带边框border的盒子
PS：配合oasis-box-header oasis-box-body oasis-box-footer使用
oasis-box-header bgnone Boolean或不赋值 ---- ---- 盒子标题区域,通过bgnone设置标题区域是否需要背景色
